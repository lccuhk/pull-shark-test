# 🦈 Pull Shark Test Repository

这个仓库是用来获取GitHub Pull Shark成就的测试项目！

## 📋 关于这个项目

这是一个专门用来学习和练习GitHub Pull Request工作流的示例仓库。

## 🎯 目标

获得GitHub的Pull Shark成就徽章！🦈

## 📝 如何获得Pull Shark成就

1. 创建一个Pull Request
2. 确保它被合并
3. 再创建另一个Pull Request并合并
4. 完成！你就获得了Pull Shark成就！
