## 🚀 开始使用

### 运行示例代码

```bash

git clone https://github.com/lccuhk/pull-shark-test.git
cd pull-shark-test

# 运行示例
python examples/hello-world.py
```

### 示例代码

查看 [examples/hello-world.py](examples/hello-world.py) 了解如何使用。

```python
from examples.hello_world import hello_world, greet

print(greet("GitHub"))
```
